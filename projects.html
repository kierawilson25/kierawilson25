Project #1: Design Sytem Competetive Review

This project was one I completed during my Internship at Ameriprise Financial this summer. I did extensive research on 10 different design systems including Material Design by Google, Carbon by IBM, and Cedar by REI. I complied all of the information that I gathered into an excel spreadsheet, an indepth powerpoint, and an overview powerpoint. I then presented my findings to the User Interface Design & Development team inorder to inspire their development of Ameriprise's design system, Adel.

Project #2: Tommie Coin Wallet

This project was devloped in my Advanced Information Security class using a linux virtual machine and coding in C. The purpose of the project was to get a deeper understanding of how cryptocurrency works. We spent time studying early versions of cryptocurrency and bitcoin, we then went on to create our own version of it. The wallet used a blockchain hosted on a local server to allow a user to check their balance, pay someone, or mine for more Tommie Coins. Overall this project was very challanging but it is one of my favorite projects I worked on so far
